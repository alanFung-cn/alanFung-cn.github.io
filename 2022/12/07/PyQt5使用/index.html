环境配置

软件安装

PyqQ5基于python环境，推荐使用python3.8版本,先进行python的安装，安装的时候要勾选“将环境变量添加到Path”中选项。Vscode没有什么特别的要求，正常安装即可。

PyQt包及相关工具安装

安装PyQt相关的包，使用pip输入下面的代码

#安装PyQt5
pip install PyQt5 -i https:pypi.douban.com/simple
#安装PyQt5-tools
pip install PyQt5-tools -i https:pypi.douban.com/simple
# 注：
# 1、不用镜像的话则不需要加后面的  -i https:pypi.douban.com/simple  不过国内的镜像下载速度会更快
# 2、PyQt5是核心部件，一些必要的库会打包在里面，但是里面不包含用于界面设计的QT Designer，因此需要下载PyQt5-tools

QT Designer的VsCode配置

完成上一步骤后，软件环境基本上已经搭建好了，下面需要配置QT Designer，使得其在VsCode软件中使用更加方便。

  1. 首先，需要确定QT Designer的安装位置。
    一般来说QT Designer的位置基于python，根据不同的版本分别位于
    # PyQt 5.15.x 之前的版本
%Python_path%\Lib\site-packages|pyqt5_tools\Qt\bin\desginer.exe

# PyQt 5.15.x 以及之后的版本
%Python_path%\Lib\site-packages\qt5_applications\Qt\bin\designer.exe

# python的路径根据自己电脑来看，忘记了可以到环境变量里找到
# 一般Python默认安装位置为：
C:\Users\Admin\AppData\Local\Programs\Python
  2. 在Vscode插件里找到PYQT Integration插件并下载

  1. 配置QT Designer在PYQT Integration插件中的路径，找到插件面板里的设置，将 QT Designer Path，这个选项的路径设置为QTDesigner的路径

  1. 配置好后，就可以在VsCode的文件树里右键之后找到相应的菜单

选择的文件如果是.ui或者.py，则会有更丰富的选项，至此，PyQt5的开发环境已经搭建完成。

为了更加方便地进行页面的构建,QT Designer提供了图形化设计页面的方式，通过对于各种组件的拖拽就可以简单快捷地创建页面，比如下图的页面：

创建好之后将页面文件保存到制定文件夹（其格式为.ui），在Vscode中打开，会发现这个格式的文件是类似xml的

这种格式的文件python无法直接解析，因此需要使用前面安装好的插件将其转化成python文件：

想要打开这个程序就需要额外再创建一个文件来调用此文件，如下

import PyQt5
import sys
from PyQt5.QtWidgets import QWidget, QApplication,QMainWindow
from Ui_untitled import Ui_MainWindow

# python的类继承是放在括号里的，继承顺序从左往右
class MyWindow(QMainWindow,Ui_MainWindow):
    # __init__是类的构造函数，固定名称
    def __init__(self, parent=None):
        # 调用父类的初始化
        super(MyWindow,self).__init__(parent)
        # 调用.ui文件转化出的.py文件中的方法，直接在这里写也是一样的
        self.setupUi(self)

# 固定格式
if __name__=="__main__":
    app=QApplication(sys.argv)
    myWin=MyWindow()
    myWin.show()
    sys.exit(app.exec_())

页面示例

功能简述

简单的 分：秒 计时器，通过两个Spin box控件设置时间，最大为59：59。

开始按钮：点击开始按钮开始计时，开始按钮变为暂停，再次点击暂停计时，暂停按钮变为继续。

还原按钮：点击还原按钮，计时器归零

主Python文件

# 文件名：Time-meter.py
from PyQt5.QtWidgets import QApplication,QMainWindow
from PyQt5.Qt import QTimer
from Ui_untitled import Ui_MainWindow
import sys

class Time_meter(QMainWindow,Ui_MainWindow):
    # 总倒计时的时间
    timeTotal=0
    # 0是未开始或正在暂停中 1是正在倒计时
    timeIsStartFlag=0
    # 0不是第一次 1是第一次
    timeIsFirst=0
    s=0

    def __init__(self):
        super(Time_meter,self).__init__()
        self.setupUi(self)

        # 初始化设置第一次的标志按钮
        self.timeIsFirst=1

        # 连接函数
        self.btn_Start.clicked.connect(self.start)
        self.btn_Reload.clicked.connect(self.reload)

        # 设置计时器
        self.timer=QTimer(self)
        # self.timer.setInterval(1000)
        self.timer.timeout.connect(self.onTimeOut)
        
    # 还原按钮
    # 类里的方法一定要传入一个参数self，当然取其他名字也可以，但是一般还是按照约定俗成取self
    def reload(self):
        self.spin_minute.setValue(0)
        self.spin_second.setValue(0)
        self.timeIsFirst=1
        self.timeTotal=0
        self.timeIsStartFlag=0
        self.btn_Start.setText("开始")
        self.timer.stop()
        
    # 开始/暂停按钮
    def start(self):
        # 第一次点击校验
        if self.timeIsFirst==1:
            self.timeTotal=int(self.spin_minute.value())*60+int(self.spin_second.value())
            self.timeIsFirst=0
            self.timeIsStartFlag=1
            self.timer.start(1000)
            return

        if self.timeIsStartFlag==0:
            self.timeIsStartFlag=1
        else:
            self.timeIsStartFlag=0


    def onTimeOut(self):
        if self.timeIsStartFlag==1:
            self.btn_Start.setText("暂停")
            if self.timeTotal!=0:
                self.timeTotal-=1
                minute=self.timeTotal//60
                second=self.timeTotal%60
                self.spin_minute.setValue(minute)
                self.spin_second.setValue(second)
            else:
                self.btn_Start.setText("开始")
                self.timeIsFirst=1
        else:
            self.btn_Start.setText("继续")          

if __name__=='__main__':
    app=QApplication(sys.argv)
    myApp=Time_meter()
    myApp.show()
    sys.exit(app.exec_())

使用PyInstaller打包成exe

pyinstaller 安装

# 安装pyinstaller，使用国内的源
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple pyinstaller

pyinstaller 打包

# 打包
# -i是图标参数，放在项目文件同文件夹即可
pyinstaller -F -w -i='clock.ico' Time-meter.py
# -F：打包成单文件   -D打包成动态库

调用upx打包

# upx需要先下载下来
# --upx-dir='D:\12python\pyqt\upx-4.0.0-win64' 这个参数的值地址即upx.exe的保存地址
pyinstaller -F -w -i='clock.ico' --upx-dir='D:\12python\pyqt\upx-4.0.0-win64' Time-meter.py
